Project Description:


firewatchBC is in an interactive web visualization for monitoring wildfires within British Columbia. The application was created as an interdisciplinary Computer Science and Geography project for CSc 497 at the University of Victoria. The firewatchBC map allows users to view the current locations and sizes of wildfires in BC, as well as viewing the maximum and minimum temperature and precipitation for select fire locations (I hope to add this feature for ALL fire locations soon).

The application is built on the Flask micro web-framework for python3 and deployed using Docker and Heroku. The overlays shown in the firewatchBC Map were created using the Folium package for visualization and GeoPandas for data management. The matplotlib package was used to create the temperature and precipitation line graphs.

Data:

Environment Canada Weather Station Locations: The weather station locations data was from the Government of Canada. This data was difficult to work with as it required downloading each dataset on a station by station basis. Fortunately, a dataset provided by Brock University organized the stations on a provincial level.

Environment Canada Weather Station Climate Data: The temperature and precipitation readings were obtained from the Pacific Climate Impacts Consortium (PCIC) Data Portal. I was unable to successfully link the PCIC data with the location data in all instances. The ID values given by PCIC did not seem to match the ID values given by Environment Canada. This is something to look further into in the future.

Current Fire Locations: The fire location data was obtained from the Government of British Columbia Open Data catalogue. This dataset is frequently updated.
